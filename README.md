# Java MCA – Programs, Notes & Reports

This repository is used by **MCA students** to submit Java programs, notes, and practical reports.

Each student must create their own folder and upload their work inside it.

---

## 📌 Repository Workflow

Follow these steps carefully:

1. **Fork this repository**
2. **Clone your fork to your system**
3. **Create a folder with your name**

Example:

```
Java-MCA
 ├── Kundan-Tripathi
 │    ├── AddTwoNumbers.java
 │    ├── Factorial.java
 │    └── Report.pdf
```

4. Add your **Java programs, notes, and reports** inside your folder.
5. Commit and push your changes to your fork.
6. Create a **Pull Request** to this repository.

---

## ⚠️ Important Rules

* Do **NOT upload files outside your folder**
* Do **NOT modify other students' folders**
* Use **clear file names** for programs
* Push updates regularly
